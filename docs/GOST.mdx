# GOST (GO Simple Tunnel) / 無感 Gemini proxy

GO Simple Tunnel 係一個好簡單嘅 proxy，
佢可以依照 url address，然後安排該 request 經指定 proxy route 出去。

例子:
我想 gemini traffic route 去 DE 嘅 VPS (a.k.a. 扮係 DE 度上網，攞  DE 嘅 IP)
其餘 traffic 經我屋企普通 ISP

因為整個過程都係 router + docker 入面進行，用戶唔需要做額外設定，所以叫透明代理。
